# Recent Sessions

Session logs that are 7+ days old but not yet archived.

## Purpose

This directory contains session logs that have been moved from `current/` after 7 days. These sessions are kept here before being archived into monthly zip files.

## Lifecycle

1. Sessions are moved here from `current/` after 7 days
2. Monthly archives are created in `archived/` (format: `YYYY-MM.zip`)
3. Very old sessions (90+ days) can be cleaned up

See `../README.md` for full session management documentation.
