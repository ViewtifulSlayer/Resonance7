# Load Foundation

Load and apply the Resonance 7 foundation configuration.

## Action

When this command is invoked:
1. Read the file `library/agent_foundation.json`
2. Apply the core philosophy, behavioral guidelines, and development protocols
3. Acknowledge that the foundation has been loaded
4. Summarize key protocols that are now active

## What It Loads

- Core philosophy (Human-AI Collaborative Synergy, Knowledge Persistence)
- Essential behavior guidelines (ethical standards, communication protocols)
- Development protocols (web_first, file_safety, command_execution)
- Workspace architecture understanding

## When to Use

- After context summarization
- When starting a new session
- If agent behavior needs to be reset to foundation standards
- When protocols or guidelines need to be re-established
