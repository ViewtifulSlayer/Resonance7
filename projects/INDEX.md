# Project Index (Template)

This file is a **workspace-local index** of workspaces under `projects/`. It is intended as a **human-readable** overview of the projects in the directory.

- Paths are relative to `projects/` (e.g. `projects/example_project/README.md`).
- Example groupings below are **illustrative only**. Customize them for your own workspace and projects.
- To keep this file reusable, avoid hard-coding private or highly specific session details when sharing the template.
