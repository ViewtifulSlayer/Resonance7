# Tests

Test files and test utilities.

## Purpose

This directory contains all test files for the project, including:
- Unit tests
- Integration tests
- End-to-end tests
- Test fixtures and utilities

## Structure

Organize tests to mirror the source code structure for easy navigation:
- `test_*.py` - Test files following pytest conventions
- `fixtures/` - Test data and fixtures
- `helpers/` - Test utility functions

## Running Tests

Use pytest to run tests:
```bash
pytest tests/
```
