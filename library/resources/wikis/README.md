# Wikis

Knowledge base databases extracted from wikis and other documentation sources.

## Structure

Each wiki knowledge base is stored in its own subdirectory with:
- SQLite database file (`*_knowledge_base.db`)
- Index documentation (`knowledge_base_index.md`)

## Access

Use the MCP SQLite Server for direct database access. See `library/tools/mcp_sqlite_server/README.md` for setup instructions.
